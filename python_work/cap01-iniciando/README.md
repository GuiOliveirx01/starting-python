# Capítulo 1 - Iniciando

## Sobre
Este capítulo é baseado no livro Curso Intensivo de Python, de Eric Matthes.

Aqui eu aprendi os conceitos fundamentais de Python.

## Conceitos aprendidos
* o que é Python
* instalar Python em meu sistema
* instalar um editor de texto
* executar códigos Python em uma sessão de terminal

## Exercícios
* hello_world.py → exibe uma mensagem

## O que eu entendi
Aprendi que Python é uma das linguagens de programação mais usadas do mundo graças a sua facilidade de aprender e sua grande versatilidade, a como instalar Python em meu sistema através do site oficial de Python, como instalar e usar um editor de texto para facilitar a escrita de códigos Python, e como executar códigos Python em sessões de terminal.
