# Capítulo 2 – Variáveis e tipos de dados simples

## Sobre
Este capítulo é baseado no livro Curso Intensivo de Python, de Eric Matthes.

Aqui eu aprendi a trabalhar com variáveis e seus tipos.

## Conceitos aprendidos
* criação de variáveis
* resolver erros de nome e de sintaxe
* manipulação de strings
* métodos de formação
* manipulação de ints e floats
* escrever comentários
* aprendi o Zen de Python

## Exercícios
* hello_world.py → armazena uma mensagem em uma variável
* simple_message.py & simple_messages.py → exibe mensagens simples
* name.py & name_cases.py → formata strings
* apostrophe.py → resolve um erro de sintaxe
* birthday.py → exibe mensagens com valores númericos
* comment.py → mostra que comentários são ignorados na saída do terminal

## O que eu entendi
Aprendi que variáveis são como "caixas" onde podemos guardar valores para serem usados ou alterados quando quisermos, e que existem tipos de valores e alguns deles podem ser formatados. Também aprendi a resolver erros como nomes errados ou uso errado de aspas, a como escrever comentários explicativos e aprendi de manter meu código o mais simples possível.

## Dificuldades 
Tive dificuldades em interpretar os princípios do Zen de Python, mas após ver exemplos práticos desses princípios consegui entende-los.
