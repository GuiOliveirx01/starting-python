# Starting Python

🇧🇷 Português | [🇺🇸 English](README.md)

Este repositório documenta minha jornada de aprendizado em Python por meio de exercícios práticos e materiais de estudo desenvolvidos durante a exploração dos fundamentos da programação.

O principal objetivo deste projeto é construir uma base sólida em Python, aprimorar a lógica de programação e desenvolver boas práticas de desenvolvimento por meio da prática constante.

## 📚 Sobre o Repositório

Este repositório contém exercícios e exemplos abordando conceitos fundamentais da linguagem Python. Cada pasta é dedicada a um tópico específico estudado durante o processo de aprendizado.
