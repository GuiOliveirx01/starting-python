# Python Lists

This folder contains exercises and small programs created while learning how to work with lists, one of the most important and versatile data structures in Python.

## 📚 Practiced Concepts

* Creating and modifying lists
* Accessing items by index
* Iterating through lists with `for` loops
* String manipulation with lists
* List slicing and copying
* Working with empty lists
* Organizing and sorting data

### List Methods

* `.append()`
* `.insert()`
* `.pop()`
* `.remove()`
* `.sort()`
* `.reverse()`

### Built-in Functions

* `min()`
* `max()`
* `sum()`
* `len()`
* `sorted()`

### Additional Concepts

* Removing items with `del`
* Modifying values inside lists
* Automated repetition with loops
* Creating slices and copying lists

## 📂 Files

* `names.py` → Iterating through lists with `for`

* `cars.py` → Accessing specific items in a list

* `languages.py` → Adding, removing, and organizing items

* `pizzas.py` → Loops and personalized messages with lists
