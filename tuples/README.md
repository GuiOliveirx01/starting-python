# Python Tuples

This folder contains exercises created while learning how to work with tuples and immutable data structures in Python.

## 📚 Practiced Concepts

* Creating tuples
* Accessing tuple elements
* Tuple immutability
* Iterating through tuples with `for` loops
* Reassigning tuple variables

## 🎯 Learning Outcomes

By completing these exercises, I practiced:

* Working with immutable collections
* Accessing and displaying tuple data
* Iterating through tuples using loops
* Understanding when tuples are more appropriate than lists

Practice the use of tuples and understand their differences compared to lists and other Python data structures.
