# Python Variables

This folder contains exercises and small programs created while learning the fundamentals of variables, strings, and basic operations in Python.

## 📚 Practiced Concepts

* Creating variables
* Printing messages with `print()`
* Using `f-strings`
* String methods:

  * `.title()`
  * `.upper()`
  * `.lower()`
  * `.strip()`
* Line breaks and formatting
* Basic mathematical operations
* Variable reassignment

## 📂 Files

* `hello_world.py` → Introduction to output with `print()`
* `simple_message.py` → Storing messages in variables
* `simple_messages.py` → Variable reassignment practice
* `apostrophe.py` → Using apostrophes in strings
* `name_cases.py` → Text manipulation and formatting
* `basic_math.py` → Basic mathematical operations
* `name_formatting.py` → Using the `.title()` method

## 🎯 Learning Outcomes

By completing these exercises, I practiced:

* Storing and updating data in variables
* Manipulating and formatting strings
* Performing basic calculations
* Writing clearer and more readable code

## 🚀 Goal

Practice the fundamentals of Python and develop increasingly organized, readable, and maintainable code.
