# Python Loops

This folder contains exercises created while learning how to use loops and repetition structures to automate tasks and process data efficiently in Python.

## 📚 Practiced Concepts

* `for` loops
* Using the `range()` function
* List comprehensions
* Numeric sequences
* Automated repetition
* Generating lists with loops
* Basic mathematical calculations with loops
* Iterating through collections

## 📂 Files

* `range_loop.py` → Counting using `range()`
* `odd_numbers.py` → Generating odd numbers with `range()`
* `multiples.py` → Working with multiples of 3
* `cubes.py` → Generating cube values using loops
* `list_comprehension.py` → Creating lists with list comprehensions
* `one_million.py` → Exploring large numeric ranges and calculations

## 🎯 Learning Outcomes

By completing these exercises, I practiced:

* Automating repetitive tasks
* Working with numeric sequences
* Iterating through data efficiently
* Generating and manipulating lists
* Applying loops to solve simple programming problems

## 🚀 Goal

Practice repetition structures and develop the ability to automate tasks and process data using loops in Python.
